🔒PrivatlivData, Firebase og brugerrettigheder

Privatlivspolitik

Work4it behandler konto-, profil-, trænings-, kropsmålings- og medlemskabsdata for at kunne levere appens funktioner.

Når brugeren er logget ind, gemmes understøttede data i Firebase/Firestore. Lokale data bruges som cache/fallback.

Brugeren kan anmode om eksport og sletning af egne data. Denne tekst skal juridisk gennemgås før kommerciel lancering.
